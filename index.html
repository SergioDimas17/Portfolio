Proyecto Presupuestos

Este proyecto es una aplicación web que te permite realizar un seguimiento de tus ingresos y gastos. La aplicación te permite agregar nuevos ingresos y gastos, editar los ingresos y gastos existentes, y eliminar los ingresos y gastos que ya no necesitas. Para agregar un nuevo ingreso, debes proporcionar una descripción del ingreso y el valor del ingreso. Para agregar un nuevo gasto, debes proporcionar una descripción del gasto y el valor del gasto. Puedes editar los ingresos y gastos existentes cambiando la descripción o el valor del ingreso o gasto. También puedes eliminar los ingresos y gastos que ya no necesitas.La aplicación muestra un resumen de tus ingresos y gastos en el encabezado de la aplicación. El resumen muestra el presupuesto total, el total de ingresos y el total de gastos. La aplicación web cuenta con conexion con una base de datos que permite almacenar todos los ingresos e egresos que el usuario desee agregar.

Pagina La Cava

El proyecto es una página web estática para un restaurante llamado "La Cava". La página web está diseñada para informar a los visitantes sobre el restaurante, sus platos y bebidas, y sus horarios de atención. La página web comienza con un encabezado que incluye el logotipo del restaurante y un menú de navegación. El menú de navegación permite a los visitantes navegar por las diferentes secciones de la página web. La descripción del restaurante explica que el restaurante ofrece una experiencia culinaria única, con platos elaborados con ingredientes frescos y de temporada. La sección "Formulario de reservación" incluye un formulario para que los visitantes puedan reservar una mesa en el restaurante. La sección "Información de contacto y enlaces" incluye información de contacto del restaurante, enlaces a redes sociales y enlaces a información útil.

Buscador del Clima

Este proyecto web permite a los usuarios obtener información meteorológica de una ciudad y país específicos. El propósito general del proyecto es ofrecer una interfaz sencilla para que cualquier persona, independientemente de sus conocimientos en programación, pueda conocer el clima actual de una ubicación de su elección. En términos prácticos, el proyecto se presenta como una página web con un formulario en el que los usuarios pueden ingresar el nombre de la ciudad y el país para el cual desean obtener información meteorológica. El formulario es intuitivo, con campos de entrada para ciudad y país, y un botón para enviar la solicitud. Cuando el usuario envía el formulario, el código JavaScript entra en acción y utiliza una API de OpenWeatherMap para recuperar datos meteorológicos en tiempo real de la ubicación proporcionada. Esta API actúa como un servicio externo que provee información
